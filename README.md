# Friendlier Fire (OpenMW)

Rebelling followers begone!

*For MWSE alternative check out [No More Friendly Fire](https://www.nexusmods.com/morrowind/mods/48801) by Celediel.*

## What It Does

- Prevents any followers from aggroing at you. Includes:
  - NPC/creature followers
  - Player's summons
  - Follower's summons
- Stops followers from fighting each other [NEEDS TESTING]
- Reduces physical player-to-follower damage and vice versa (configurable)

## What It Doesn't (yet)

- Affect spells/enchantments

## Requirements

**Load dependencies before this mod!**

- Follower Detection Util (link will be added later)
